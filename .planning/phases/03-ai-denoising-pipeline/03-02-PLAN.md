    NoiseReductionService is a Swift actor. Key implementation rules:

    1. MODEL LOADING — lazy, cached, MLComputeUnits.all:
       ```swift
       actor NoiseReductionService {
           private var _model: DeepFilterNet3?
           private func model() throws -> DeepFilterNet3 {
               if let m = _model { return m }
               let config = MLModelConfiguration()
               config.computeUnits = .all
               let m = try DeepFilterNet3(configuration: config)
               _model = m
               return m
           }
       }
       ```

    2. STREAMING INFERENCE — chunk → infer → write → discard (NOT accumulate all frames):
       - Open AVAudioFile(forReading: inputURL) with processingFormat
       - Open AVAudioFile(forWriting: outputURL) with Float32 48kHz stereo non-interleaved settings
       - Iterate 480-sample chunks via inputFile.read(into: readBuffer)
       - Zero-pad LAST chunk if frameLength < 480 (fill MLMultiArray beyond frameLength with 0.0)
       - Run model.prediction(input_frame:hidden_state_in:) with await — exact param names from generated Swift wrapper
       - Write output chunk IMMEDIATELY to outputFile — do NOT store in array
       - Thread hiddenStateOut → next iteration hiddenStateIn
       - yield progress via continuation.yield(Float(processedFrames) / Float(totalFrames))

    3. BLEND FUNCTION — public, free function or static method accessible from ViewModel:
       ```swift
       func blend(original: [Float], denoised: [Float], intensity: Float) -> [Float]
       ```
       Use vDSP.add(multiplication:multiplication:) Swift overlay (Accelerate framework, iOS 13+).
       Assert original.count == denoised.count.

    4. CANCELLATION — check Task.isCancelled inside the inference loop. Break and call continuation.finish() on cancel.

    5. ERROR HANDLING — if model loading fails (mlpackage missing), throw a descriptive error with the bundle resource name.

    6. iOS 17 CONSTRAINT — add comment: "MLState requires iOS 18+. This service uses explicit RNN state I/O. Do not migrate to MLState until deployment target is iOS 18+."
