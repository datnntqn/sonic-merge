**MixingStationViewModel.swift — Add duplicate detection (D-10/D-11):**

In `performImport(urls:)`, add a duplicate check BEFORE the normalization call (inside the `for url in urls` loop, after `url.startAccessingSecurityScopedResource()` and before `try await normalizationService.normalize(...)`):

```swift
let displayName = url.deletingPathExtension().lastPathComponent
let isDuplicate = clips.contains { $0.displayName == displayName } ||
                  newClips.contains { $0.displayName == displayName }
guard !isDuplicate else { continue }
```

This checks both existing SwiftData clips AND clips being imported in the current batch. If a match exists, silently skip the URL (no error appended per D-10).

**SonicMergeApp.swift — Add scenePhase pending import handler (overrides D-09 per RESEARCH.md Pitfall 1):**

1. Add a `@Environment(\.scenePhase) private var scenePhase` property to `SonicMergeApp`.

2. Extract the `MixingStationViewModel` to a `@State` property so it can be referenced in the `.onChange` modifier:
```swift
@State private var viewModel: MixingStationViewModel?
```

3. In the `WindowGroup` body, after `.modelContainer(modelContainer)`, add:
```swift
.onChange(of: scenePhase) { _, newPhase in
    guard newPhase == .active else { return }
    let defaults = UserDefaults(suiteName: AppConstants.appGroupID)
    guard let filename = defaults?.string(forKey: "pendingImportFilename") else { return }
    defaults?.removeObject(forKey: "pendingImportFilename")
    guard let clipsDir = try? AppConstants.clipsDirectory() else { return }
    let fileURL = clipsDir.appending(path: filename)
    guard FileManager.default.fileExists(atPath: fileURL.path) else { return }
    viewModel?.importFiles([fileURL])
}
```

4. Restructure the WindowGroup body to keep a reference to the viewModel. The pattern:
```swift
@State private var viewModel: MixingStationViewModel?

var body: some Scene {
    WindowGroup {
        Group {
            if let viewModel {
                MixingStationView()

            }

        .onAppear {

        .onChange(of: scenePhase) { _, newPhase in

    }
    .modelContainer(modelContainer)
```
