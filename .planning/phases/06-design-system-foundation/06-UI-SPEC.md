# Phase 6 — UI Design Contract

> Visual and interaction contract for the design system foundation. Generated by gsd-ui-researcher, verified by gsd-ui-checker.
> Platform: SwiftUI / iOS native. shadcn is not applicable.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (SwiftUI native — no component library framework) |
| Preset | not applicable |
| Component library | SwiftUI built-ins + custom (SquircleCard, PillButton) |
| Icon library | SF Symbols (system icons throughout existing codebase) |
| Font | SF Pro (system default via `.font(.system(...))`) |

**Pattern:** UIColor stored in `SonicMergeSemantic` struct, wrapped with `Color(uiColor:)` at SwiftUI call sites. Follow this pattern for all new color token slots. No new EnvironmentKey is needed — extend the existing `SonicMergeSemanticKey`.

---

## Spacing Scale

Declared values (multiples of 4, in SwiftUI points — 1pt = 1px at 1x):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4pt | Icon gaps, tight inline padding, waveform bar gaps |
| sm | 8pt | Compact element spacing, VStack inter-item gaps |
| md | 16pt | Default element padding, HStack spacing in cards |
| lg | 24pt | Section padding, card internal padding |
| xl | 32pt | Layout gaps between major sections |
| 2xl | 48pt | Navigation header clearance, major section breaks |
| 3xl | 64pt | Page-level vertical rhythm |

Exceptions:
- Touch targets for icon-only buttons: minimum 44pt x 44pt (iOS HIG minimum) — source: existing `ClipCardView` preview button uses 10pt padding around icon, producing 44pt tap area.
- `LocalFirstTrustStrip` internal padding: 12pt (existing; retain for glassmorphism restyle).
- `SonicMergeTheme.Radius.card`: update from 12pt to **24pt** for SquircleCard (DS-02 requirement). `SonicMergeTheme.Radius.chip`: retain 8pt for pill-shaped gap controls.

---

## Typography

All sizes in SwiftUI dynamic type system roles. Weights are exact (`fontWeight` modifier).

| Role | SwiftUI Role | Weight | Line Height |
|------|-------------|--------|-------------|
| Body | `.body` (~17pt default) | `.regular` (400) | 1.5 (system default) |
| Label / Caption | `.caption` (~12pt default) | `.regular` (400) | 1.3 |
| Subheadline / Card title | `.subheadline` (~15pt default) | `.semibold` (600) | 1.3 |
| Header / Display | `.title3` (~20pt default) | `.semibold` (600) | 1.2 |

**Rule:** Exactly 2 weights in use — `.regular` (400) and `.semibold` (600). No `.bold`, `.heavy`, or other weights.

**Existing pattern to preserve:** `ClipCardView` uses `.system(.body, design: .default, weight: .medium)` — Phase 7 will update these to `.semibold`; Phase 6 does not touch existing views.

## Color

### v1.1 Palette (replaces v1.0 hard-coded values)

| Role | Light Mode | Dark Mode | SwiftUI Usage |
|------|-----------|-----------|---------------|
| Dominant — background (60%) | `#FBFBFC` (off-white) | `#000000` (pure black) | `surfaceBase` token |
| Secondary — cards / elevated (30%) | `#FFFFFF` | `#0F0F0F` (near-black card) | `surfaceSlot` / `surfaceElevated` tokens |
| Accent — Deep Indigo (10%) | `#5856D6` | `#5856D6` | `accentAction`, `accentGlow` tokens |
| AI Accent — Lime Green | `#A7C957` | `#A7C957` | `accentAI` token |
| Destructive | iOS system `.red` (`#FF3B30`) | iOS system `.red` | Destructive-only (delete clip, remove action) |

**Source:** DS-01 (REQUIREMENTS.md), D-01 through D-05 (CONTEXT.md).

### 60/30/10 Split Details

- **60% Dominant:** `surfaceBase` — fills the full-bleed NavigationStack background, ignoring safe area. Pure black in dark mode, `#FBFBFC` in light mode.
- **30% Secondary:** `surfaceSlot` / `surfaceElevated` — fills SquircleCard backgrounds, modal sheet backgrounds, and the glassmorphism header layer. Near-black `#0F0F0F` in dark mode (distinct from `#000000` to give card lift); `#FFFFFF` in light mode.
- **10% Accent:** `accentAction` (Deep Indigo `#5856D6`) — reserved exclusively for:
  - PillButton fill / inner glow highlight
  - Glassmorphism header "Private by Design" glow ring
  - Trust strip icon (`lock.shield.fill`)
  - Active waveform bars (Mixing Station — Phase 7 scope, token defined here)
  - Navigation title tint (if overridden)
- **AI Accent:** `accentAI` (Lime Green `#A7C957`) — reserved exclusively for:
  - Denoising progress indicator (Cleaning Lab — Phase 8 scope, token defined here)
  - Noise suppression slider fill
  - AI Orb pulsating glow (Phase 8)
  - AI feature action button highlights

### Accent Reservation (strict)
Deep Indigo is **not** used for body text, secondary text, decorative lines, or disabled states. Lime Green is **not** used outside AI-identified features.

### New Token Slots to Add to `SonicMergeSemantic`

| Slot | Type | Description |
|------|------|-------------|
| `accentAI` | `UIColor` | Lime Green `#A7C957` — same in light and dark |
| `accentGlow` | `UIColor` | Deep Indigo `#5856D6` — used for glow shadows and ring borders |
| `surfaceCard` | `UIColor` | Squircle card fill — `#FFFFFF` light / `#0F0F0F` dark |
| `surfaceGlass` | `UIColor` | Glassmorphism header tint — `#FBFBFC` at 0.6 opacity light / `#000000` at 0.7 opacity dark |

**Migration note (D-01):** Existing `accentAction` becomes Deep Indigo (replacing `#007AFF`). `accentWaveform` also becomes Deep Indigo. The v1.0 charcoal dark palette is replaced wholesale (D-02).

## Component Contracts

### SquircleCard

- Corner radius: 24pt continuous (`RoundedRectangle(cornerRadius: 24, style: .continuous)`)
- Background: `surfaceCard` token (opaque) or `.ultraThinMaterial` when `glassEnabled: true`
- Shadow (default): `Color.black.opacity(0.10)` radius 16, y-offset 6 — both modes
- Shadow (elevated / drag state): `Color(uiColor: semantic.accentGlow).opacity(0.25)` radius 24, y-offset 10 — Phase 7 scope, token reserved here
- Border: 1pt stroke `Color(uiColor: semantic.accentGlow).opacity(0.18)` when `glassEnabled: true`
- Padding internal: 16pt (md)
- API parameters: `glassEnabled: Bool = false`, `glowEnabled: Bool = false`, `content: () -> Content`
- `reduceTransparency` fallback: when `.ultraThinMaterial` is active and `UIAccessibility.isReduceTransparencyEnabled`, replace material with solid `surfaceCard` color

### PillButton (custom `ButtonStyle`)

- Shape: `Capsule()` (infinite corner radius)
- Background fill: `accentAction` (Deep Indigo)
- Inner glow: subtle white shimmer — `LinearGradient` from `Color.white.opacity(0.25)` top to `Color.white.opacity(0)` bottom, overlaid at 60% height
- Label: `.subheadline.weight(.semibold)`, foreground `.white`
- Padding: 12pt vertical, 24pt horizontal
- Press animation: `scaleEffect(0.96)` on press with `.spring(response: 0.25, dampingFraction: 0.6)`
- Haptic: `.sensoryFeedback(.impact(.light), trigger: isPressed)` — fires on press down
- Disabled state: `accentAction` at 0.35 opacity, no inner glow, no haptic
- Minimum touch target: 44pt height enforced via `.frame(minHeight: 44)`

### Glassmorphism Header (LocalFirstTrustStrip restyle)

- Background: `.ultraThinMaterial` with `accentGlow` tint overlay at 0.08 opacity
- Border: 1pt stroke `accentGlow` at 0.30 opacity (continuous rounded rect, radius 16pt)
- Glow shadow: `Color(uiColor: semantic.accentGlow).opacity(0.20)` radius 12, y-offset 0 (ambient ring effect)
- Content: `lock.shield.fill` SF Symbol (`.title3`, `accentGlow` color) + title `.subheadline.semibold` + subtitle `.caption.regular`
- `reduceTransparency` fallback: replace `.ultraThinMaterial` with solid `surfaceCard`
- Scroll behavior: pinned at top of `NavigationStack` toolbar area (not scrolling with content)

## Accessibility Contract

These rules apply to all Phase 6 components and propagate to phases 7–9:

| Rule | Enforcement |
|------|------------|
| Glass surfaces + `isReduceTransparencyEnabled` | Replace `.ultraThinMaterial` with solid `surfaceCard` color — both SquircleCard and glassmorphism header |
| Animated components + `isReduceMotionEnabled` | PillButton press scale animation is removed; static tint change only. No animated transitions in Phase 6 primitives. |
| Text contrast on glass | Minimum 4.5:1 ratio required. `textPrimary` (`#1C1C1E` light / near-white dark) on `surfaceCard` meets this. Deep Indigo `#5856D6` on white: 4.54:1 — passes at subheadline+ sizes |
| Touch targets | 44pt minimum enforced in PillButton and any icon-only buttons |
| Dynamic Type | All text uses SwiftUI system roles (`.body`, `.caption`, `.subheadline`, `.title3`) — scales automatically |
